## Rappels essentiels

- changements minimaux, incrementaux et reversibles
- aucun secret dans les fichiers versionnes
- services Compose courants : `db`, `backend`, `frontend`
- backend sous `/api`
- auth principale : JWT SimpleJWT
- `whoami` JWT : `/api/whoami/` et alias `/api/auth/whoami/`
- logout JWT : `/api/auth/jwt/logout/`
- compat session legacy : `/api/auth/session/*`
- aliases historiques `/api/csrf/`, `/api/login/`, `/api/logout/` deprecies
- `VITE_API_BASE=/api`
- frontend sans URL absolue backend
- cle privee locale stockee en `IndexedDB`, pas en clair dans `localStorage`

## Quand tu proposes un correctif

1. verifier qu'il respecte `INVARIANTS.md`
2. ne pas reintroduire un ancien contrat documente ailleurs dans `docs/`
